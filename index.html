About this site

This site was made to practice html, css and js. And also to figure out what can be done with modern CSS.

I'm trying to use as many fun features of css as possible (that are useful) including one's that are not yet supported by all browsers (sorry firefox users).

Some features that I have used:

  • conic gradients
  • animations/ keyframes
  • the property tag
  • grid layout
  • flexbox

P.S. Some test stuff is around this site, don't worry too much
